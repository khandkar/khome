--------
- Laptops:
    - Purism Librem 15 v4
    - ThinkPad T420

    - ThinkPad T570
- Monitors:

- Dictionary:

- File browsing:
    - `ls` and `tree` (primary)
    - `pcmanfm` (secondary)
    - `ranger` (occasional)

- IRC:
    - `ii`
    - `ii-tools`
- Multimedia:
    - Image:
        - Editing:
            - `gimp`
        - Viewing:
            - `sxiv` (primary)

            - `ripit` (rip CDs)
            - `mpd`
            - `mpc`
            - `ncmpcpp`
            - `ncmpc` (to delete from playlist, which doesn't work in `ncmpcpp`)
        - Streaming:
            - [soma fm](http://somafm.com/) (primary) (via `mpd`)
            - `spotify` (occasional)
    - Video:
        - `mpv`
        - `mplayer`
        - `vlc`
- PDF:
    - `okular`
    - `mupdf`
    - `pdftotext`
- Search:
    - `locate`

    - `grep`
- Terminal:
    - `st`
    - `tmux`
    - `zsh`
- Text editing:
    - `vim`
    - `gvim`
- Web browsing:
    - `qutebrowser` (primary)

    - `elinks` (occasional)

- X
